What is this board?

A real-time honesty board: each check verifies that one workflow invariant actually holds right now — docs in sync, rules enforced, hooks live, no drift. The board is green only when the guarantee is real, not when prose claims it. This is the make-real goal: surface gaps immediately so every "done" claim is backed by evidence.
